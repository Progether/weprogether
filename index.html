What is Progether?

Progether is centered around one simple belief: It's enjoyable and effective to learn programming by collaborating on projects with others. So, what are you waiting for?

Github

Our version control tool of choice is the venerable Git. To publicly host all of our project code, we use Github, a web-based hosting service for projects that use the Git version control system. Head on over to Github to get started!
